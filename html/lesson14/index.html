Lesson 13

Нужно разбивать на логические кусочки содержимое веб странички например тегами div, но в современном html принято использовать

  • nav
  • header
  • main
  • aside
  • section
  • article
  • footer
    • Дополкие нительные семантические независисмы теги
  • figure - для отображнеия контента напримре типа картинки img или циаты bloquote
  • details - дополнительные сведения которые юзер может просматривать или скрывать
  • dialog - определяет диалогвое окно
